## 새 게시글 작성

### POST /posts/post

#### Content-Type: application/json

#### Authorization: \{\{access\_token}}

#### Refresh-Token: \{\{refresh\_token}}

```txt
{
  "title": "{{$random.alphanumeric(57)}}",
  "content": "<p>{{$random.alphanumeric(145)}}</p> </br> {{$random.alphanumeric(157)}}"
}
```

### 반환값

```txt
{
 "code" : 201
 "message" : "게시글 작성 완료"
}
```

## 게시글 파일 업로드

### POST http://localhost:8080/posts/post/file-upload?

### &#x20;   postId=\{\{postID}}

#### Content-Type: multipart/form-data; boundary=WebAppBoundary

#### Authorization: \{\{access\_token}}

#### Refresh-Token: \{\{refresh\_token}}

```txt
--WebAppBoundary
Content-Disposition: form-data; name="file"; filename="cat.jpg"
Content-Type: image/jpeg
< path/to/some.jpg
```

***

### 반환값

```txt
{
 "code" : 200
 "message" : http:{{minioURL}}/board//post/{{imageURL}}
}
```
